Personal website scaffold

Files created:
- index.html — homepage
- resume.html — resume content
- projects.html — projects listing
- css/styles.css — styles
- js/main.js — small interactions

How to use

1. Open `index.html` in your browser to see the site locally.
2. Edit the HTML pages to add your real email, links, and project details.
3. To publish, push this repo to GitHub and enable GitHub Pages on the `main` branch (or use any static host).

Tips

- Replace placeholder project cards with real descriptions and links to repos/demos.
- Add a profile image in an `assets/` folder and reference it from the pages.
- Customize colors in `css/styles.css`.
